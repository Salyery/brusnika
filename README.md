# Admin Panel for Campaign Management - Vue Version

Vue 3 проект с Vuetify 3 для управления рекламными кампаниями.

## Технологии

- Vue 3
- TypeScript
- Vite
- Vuetify 3
- Pinia (для управления состоянием)
- Material Design Icons

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Предпросмотр сборки

```bash
npm run preview
```

## Структура проекта

```
src/
  ├── components/     # Vue компоненты
  ├── views/          # Страницы приложения
  ├── plugins/        # Плагины (Vuetify и др.)
  ├── stores/         # Pinia stores
  ├── types/          # TypeScript типы
  ├── styles/         # Глобальные стили
  ├── App.vue         # Корневой компонент
  └── main.ts         # Точка входа
```
